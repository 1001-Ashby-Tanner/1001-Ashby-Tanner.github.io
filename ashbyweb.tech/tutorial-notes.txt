## django tutorial ##

create django workspace
    python -m django startproject mysite
    
cd into mysite directory

run server
    python manage.py runserver (optional port number)

create app
    python manage.py startapp main

make/save migrations
    python manage.py makemigrations main

migrate changes
    python manage.py migrate

enter python shell
    python manage.py shell

exit python shell
    CTRL + Z then ENTER
    exit()

import class models to shell
    from main.models import Item, ToDoList

create ToDoList model from shell
    t = ToDoList(name="Tanner\'s List")

other functions for model objects
    t.save()
    ToDoList.objects.all()
    ToDoList.objects.get(id=1)
    ToDoList.objects.get(name="Tanner's List")
    t.item_set.all()
    t.item_set.create(text="Sleep", complete=False)
